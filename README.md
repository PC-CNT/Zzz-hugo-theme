# Zzz-hugo-theme

## なにこれ

[Hugo](https://gohugo.io/)用自作テーマです　完成度はお察しください

有名なサイト様のレイアウトを参考にして書いたら思ったよりパクりUIになってしまいました　ゆるして

## テーマ　の　つかいかた

`hugo new site site_name`

`cd site_name`

`git submodule add https://github.com/PC-CNT/Zzz-hugo-theme.git theme/Zzz-hugo-theme`

`hugo server -t Zzz-hugo-theme`
